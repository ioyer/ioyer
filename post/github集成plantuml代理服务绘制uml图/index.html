Github集成plantuml代理服务绘制uml图

2019-09-05 约 252 字 预计阅读 1 分钟 次阅读

文章目录

使用此stackoverflow讨论中描述的PlantUML 代理服务。这种方式的工作原理是，我们定义了一个远程URL，而不是传递PlantUML服务器的URL内的图表内容，

例如，可以从中获取内容。此URL可以嵌入HTML 标记或Markdown图像语法<img> / ![]()中。
要在使用GitHub时利用此功能，只需将远程URL指向存储库中PlantUML图的原始链接即可。

<http://www.plantuml.com/plantuml/proxy?src=https://raw.github.com/plantuml/plantuml-server/master/src/main/webapp/resource/test2diagrams.txt>

您可以在我的GitHub帐户的plantbuddy和techradar项目中看到此方法。

参照:Markdown native diagrams with PlantUML
